# Plaquettes Informatives des Formations

Ce dossier contient les plaquettes PDF des formations proposées par Atipik RH.

## Fichiers à déposer :

### Formation CIP
- **Nom du fichier :** `formation-cip-plaquette.pdf`
- **Description :** Plaquette informative détaillée de la formation Conseiller en Insertion Professionnelle
- **Lien sur le site :** Accessible depuis la page `/formations/cip`

### Formation FPA
- **Nom du fichier :** `formation-fpa-plaquette.pdf`
- **Description :** Plaquette informative détaillée de la formation Formateur Professionnel d'Adultes
- **Lien sur le site :** Accessible depuis la page `/formations/fpa`

## Instructions :
1. Déposez vos fichiers PDF dans ce dossier
2. Respectez exactement les noms de fichiers indiqués ci-dessus
3. Les fichiers seront automatiquement accessibles via les boutons de téléchargement sur les pages de formations
